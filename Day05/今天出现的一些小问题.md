这样一段代码：

```
import json
dict = {'id': 1, 'title': b'\xe7\xac\xac\xe4\xb8\x80\xe7\xab\xa0 \xe7\xa7\xa6\xe7\xbe\xbd'}
dup = json.dumps(dict ,ensure_ascii=False)
print(dup)
```

Python2执行输出：

```
{"id": 1, "title": "第一章 秦羽"}
```
Python3执行报错：

```
TypeError: Object of type bytes is not JSON serializable
```
